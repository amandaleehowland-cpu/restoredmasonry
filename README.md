# Restored Masonry

Website for **Restored Masonry** — historic brick and stone restoration and the
rehabilitation of buildings built before 1940. Serving the Greater Ottawa Area
and Prescott-Russell.

🔗 Live site: https://restoredmasonry.com

## Pages
- `index.html` — home
- `blog.html` — the Journal (post index)
- `what-like-its-hard.html` — blog post

## Hosting
Static HTML, no build step. Deployed automatically by Cloudflare Pages on every
push to `main`.

- Build command: _none_
- Output directory: `/`

## Editing
Each page is a self-contained HTML file (fonts load from Google Fonts; images are
embedded). To publish a change, edit the file and push to `main` — Cloudflare
redeploys within a minute.
